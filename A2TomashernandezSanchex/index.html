Guía de Iniciación a Git 1. ¿Qué es Git? Git es un sistema de control de versiones que ayuda a gestionar los cambios en proyectos de programación. 2. Comenzar con Git Si no tienes Git instalado, descárgalo e instálalo desde git-scm.com. 3. Configuración Inicial Configura tu nombre de usuario y correo electrónico en Git: bash Copy code $ git config --global user.name "Tu Nombre" $ git config --global user.email "correo@ejemplo.com" 4. Iniciar un Repositorio Para iniciar un nuevo repositorio: bash Copy code $ mkdir mi_proyecto $ cd mi_proyecto $ git init 5. Añadir y Confirmar Cambios Para añadir archivos al repositorio y confirmar cambios: bash Copy code $ git add nombre_del_archivo $ git commit -m "Mensaje del Commit" 6. Compartir en GitHub Sube tu repositorio a GitHub: bash Copy code $ git remote add origin URL_DE_TU_REPOSITORIO $ git push -u origin master 7. Recursos Adicionales Try Git: Una introducción interactiva a Git. Documentación oficial de Git: Para más detalles. Esta guía proporciona una introducción básica a Git. Personaliza la información según tus necesidades y sigue aprendiendo a medida que avanzas en tu conocimiento de Git.
